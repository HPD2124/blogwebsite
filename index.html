Implementing A* Pathfinding

I faced several challenges: ensuring my heuristic was admissible, handling map boundaries, and reconstructing the path efficiently. After adding detailed logging, I found bugs in my neighbor generation that skipped diagonal walls.

// Full A* implementation (see Pathfinding.h for definitions)
std::vector AStar(const Vector2D& start, const Vector2D& goal, const std::vector>& grid) {
    std::priority_queue, std::greater<>> open;
    std::unordered_map gScore;
    std::unordered_map cameFrom;

    int startKey = start.y * 1000 + start.x;
    gScore[startKey] = 0;
    open.push({ start, 0, Heuristic(start, goal) });

    while (!open.empty()) {
        Node current = open.top(); open.pop();
        if (current.pos.x == goal.x && current.pos.y == goal.y)
            return ReconstructPath(cameFrom, current.pos);

        for (auto& neighbor : GetNeighbors(current.pos, grid)) {
            int key = neighbor.y * 1000 + neighbor.x;
            int tentativeG = gScore[current.pos.y * 1000 + current.pos.x] + 1;
            if (!gScore.contains(key) || tentativeG < gScore[key]) {
                gScore[key] = tentativeG;
                cameFrom[key] = current.pos;
                int f = tentativeG + Heuristic(neighbor, goal);
                open.push({ neighbor, tentativeG, f });
            }
        }
    }
    return {};
}

Building the Enemy Bullet Factory

Memory leaks and null-pointer checks were my biggest headaches. I added early returns and asserts to catch failures in EntityFactoryBuild. Getting the quad mesh size right took tweaking constants until bullets appeared at the correct scale.

// EnemyBulletFactory.c full snippet
#define BULLET_COLLIDER_RADIUS 5.0f
#define BULLET_MESH_WIDTH      0.2f
#define BULLET_MESH_HEIGHT     0.2f

Enemy* EnemyBulletFactoryBuild(void) {
    Enemy* enemyBullet = calloc(1, sizeof(Enemy));
    if (!enemyBullet) return NULL;
    enemyBullet->entity = EntityFactoryBuild("./Data/EnemyBullet.txt");
    if (!enemyBullet->entity) { free(enemyBullet); return NULL; }

    Physics* physics = PhysicsCreate();
    if (physics) { EntityAddPhysics(enemyBullet->entity, physics); }

    Transform* t = EntityGetTransform(enemyBullet->entity);
    if (t) {
        CircleCollider* c = CircleColliderCreate(TransformGetTranslation(t), BULLET_COLLIDER_RADIUS, CIRCLE, BLOCK, "Enemy_Bullet");
        EntityAddCircleCollider(enemyBullet->entity, c);
    }

    Mesh* mesh = MeshCreate();
    if (mesh) {
        MeshBuildQuad(mesh, BULLET_MESH_WIDTH, BULLET_MESH_HEIGHT, 1.0f, 1.0f, "EnemyBulletMesh");
        Sprite* sprite = EntityGetSprite(enemyBullet->entity);
        if (sprite) {
            SpriteSetMesh(sprite, mesh);
            SpriteSource* src = SpriteSourceCreate();
            if (src) { SpriteSourceLoadTexture(src, 1, 1, "Enemy_Bullet.png"); SpriteSetSpriteSource(sprite, src); }
        }
    }
    return enemyBullet;
}
